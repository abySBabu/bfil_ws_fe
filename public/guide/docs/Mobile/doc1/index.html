1. Introduction

The BFIL-Pragat – Mobile app is specifically designed for CRPs (Community Resource Persons). It is an offline-first application, allowing functionality even in areas without network coverage. The designated CRP for a specific watershed can create activities, upload multiple images, and update activity statuses from New to In Progress, and finally to Approver 1, whether online or offline. Additionally, the app enables CRPs to create and manage beneficiaries. This application streamlines the creation and tracking of activities, ensuring efficient management of statuses up to Approver 1, regardless of connectivity. Furthermore, the app supports bilingual functionality, including Kannada, making it accessible to a wider range of users.

1.1 Purpose of the document.​

This guide provides detailed instructions on how to use the BFIL-Pragat Mobile application and its various features. The purpose of this user guide is to help users understand how to effectively utilize the application for creating and updating activities, adding beneficiaries, and syncing data. The guide also explains how the app supports offline functionality, allowing users to work seamlessly in areas with limited or no network connectivity. It covers key features such as creating activities with multiple images, updating activity statuses (from New to In Progress and Approver 1), and managing beneficiary data. Additionally, users will learn how to sync data with the server once network connectivity is restored, ensuring accurate and up-to-date records.

1.2 High level Features​

The following are the key features of the BFIL-Pragat Mobile application:

  1. Application login
  2. Create Activity
  3. Edit Activity
  4. Beneficiary Creation
  5. Beneficiary Detail view
  6. Bi-lingual
  7. Profile
