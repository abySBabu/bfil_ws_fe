7. Watershed Master

Image 7.1: Watershed master page

The watershed master page contains the list of all watersheds. Watersheds can be added, edited, or deleted here.

To add a watershed, click on the ‘Add Watershed’ button in the top right corner of the page. This will open a dialog with fields for the necessary data to be filled in. Mandatory fields are marked with an asterisk (*) at the end of their names. Once all mandatory fields are filled, the add button will be enabled allowing the user to add a new watershed.

To edit a watershed, click on the edit button for that watershed. This will open a dialog with fields of the selected watershed’s data filled in. The user can edit the values of the fields from here. If all mandatory fields are filled, the user can click on the update button to save their edits.

To delete a watershed, click on the delete button for that watershed. This will open a dialog asking the user to confirm that they want to delete the selected watershed
(Note: If a watershed is assigned to one or more activities, it can’t be deleted).
