8. Beneficiary Master

Image 8.1: Beneficiary master page

The beneficiary master page contains the list of all beneficiaries. Beneficiaries can be added, edited, or deleted here.

To add a beneficiary, click on the ‘Add Beneficiary’ button on the top right of the screen. This will open a dialog with fields for the beneficiary’s name, phone number, relation and relationship, and location details to be filled.Once all mandatory fields are filled, the add button will be enabled allowing the user to add a new beneficiary.

To edit a beneficiary, click on the edit button for that beneficiary. This will open a dialog with fields of the selected beneficiary’s details filled in. The user can edit the values of the fields from here. Once all mandatory fields are filled, the user can click the update button to save their edits.

To delete a beneficiary, click on the delete button for that beneficiary. This will open a dialog asking the user to confirm that they want to delete the selected beneficiary
(Note: A beneficiary can’t be deleted if they have been assigned to one or more activities).
