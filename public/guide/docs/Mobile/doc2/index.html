2. Getting Started

2.1 Installation and Prerequisites​

Ensure the Android version is 12 or higher. You can install the application via the APK.

Steps to install the APK

  1. Click on the shared APK.
  2. Select the Package Installer.
  3. Click Install to install the application.


If the application is not installed due to security restrictions, follow these steps:

“Settings” → “Search for Install Unknown Apps”

Then, select the source from which you want to install the APK, such as WhatsApp, Chrome, or Gmail. In this example, I have used Chrome. You can manage the restricted permissions of the app from which you are downloading the APK.


After enabling it, you will be able to install the application.


Mandatory Permissions After Installing the Application

2.3 Activity Creation​

Click the Add Icon to create Activity.

After clicking the Add icon, the required app permissions are

The activity creation screen will open with separate sections for Activity, Watershed, Physical, Financial, Beneficiary, and Image details of the activity.

Select Intervention type, Activity Type from dropdown.

Fill in all the necessary details to create an activity. Each button will be enabled once the required fields are completed. The created activity will be listed on the Home page. A confirmation pop-up will appear when the add button is clicked after all necessary fields are filled.

The activity avatar icon will indicate the image sync status. A yellow avatar indicates that the image is waiting to be synced, while a blue avatar denotes that the image is synced.

The activity adding steps will be the same in offline mode as well.

Synced Image

Note: To Sync the image, pull down refresh is feature to sync the image

After the data is successfully synced, a toast message will confirm that the image has been uploaded successfully.

Note: The login, logout, and image upload features won’t work in offline mode. All other functions will work in both online and offline modes.

2.4 Edit Activity​

Upon touching the activity, the activity detail’s view will open. The details’ view displays the Activity, Watershed, Physical, Financial, Beneficiary information, and the images taken during the activity creation.

The activity detail’s view includes sections such as Activity, Watershed Details, Activity Physical Details, Activity Financial Details, Beneficiary Details, History (which contains image and remarks history), and Remarks, which have been attached respectively.

The “Update” button will be enabled only when all the mandatory fields were filled.

Updating the activity to “In Progress”. Check the box to update the Status. The button will be enabled once the Remarks is filled.

The “In Progress” status will be updated in the Home Screen as well the Activity Detail view.

The updated activity will display the entered remarks and any changes made.

Updating the activity to “Approver 1”: The three statuses of the activity are “New,” “In Progress,” and “Approver 1.” The same method used to update the activity from “New” to “In Progress” can be followed to update the activity from “In Progress” to “Approver 1.”

Additionally, I am attaching an image along with the status update.

A confirmation pop-up will appear for each creation and status update, such as "In Progress" and "Approver 1."

The activity with "Approver 1" will not be shown on mobile. If there are no changes required, the web will handle the next statuses, progressing through "Approver 2," "Approver 3," up to "Approver 5," and finally "Completed." If changes are needed, "Approver 1" will reject the activity, setting its status to "In Progress," and the task will be displayed to the responsible CRP.

2.5 Beneficiary Creation​

To create Beneficiary, click on the hamburger menu at the top left to open the sidebar and select the "Beneficiary Master" option.

The Beneficiary Master screen will open.

Click on the add icon to create Beneficiary.

Beneficiary creation form will open.

Fill the necessary details to create Beneficiary. The button will be enabled only when all the mandatory fields are filled.

The confirmation pop-up will appear when a Beneficiary is created. The created Beneficiary will be displayed in the Beneficiary master screen.

2.6 Beneficiary Detail View​

To view the Beneficiary details, click on the respective Beneficiary.

2.7 Profile​

To view the Profile, Click on the hamburger and select Profile. The Profile will show the logged in user’s details.

2.8 Bi-Lingual​

The application will support multiple languages (Kannada and English). To switch languages, disable the English option in the sidebar menu for Kannada and enable it for English.


Kannada to English


English to Kannada

2.9 Change Password​

To change your password, navigate to the Profile page. The Change Password option is located at the top-right corner. You will need to enter a New Password and Confirm Password. The new password must match the confirm password. It must include at least one special character, one uppercase letter, one lowercase letter, one number, and be at least 8 characters long. After entering the new password, click Reset to apply the changes. Once the password is successfully changed, you will need to use the new password to log in after resetting.
